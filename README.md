# Alles dreht sich weiter

A progressive web app (PWA) that exclusively displays the video player for "Alles dreht sich weiter".

## Live App

Visit the app at: [https://phroggar.github.io/AllesDrehtSichWeiter/](https://phroggar.github.io/AllesDrehtSichWeiter/)

## Features

- **PWA Ready**: Can be installed on desktop and mobile devices for offline access.
- **Offline Support**: The video and app shell are cached for offline playback.
- **Clean Interface**: A focused experience with only the video player and native controls.

## How to use

Simply visit the URL above in a modern browser. To install the app on your device, use the browser's "Install" or "Add to Home Screen" option.
